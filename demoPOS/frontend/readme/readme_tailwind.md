# setup tailwind on vite

- for vite only (cannot use create react app version)
- https://tailwindcss.com/docs/guides/vite

- copy and paste InjectTailwind.tsx in src
- In main.tsx,  
  import InjectTailwind from './InjectTailwind'
  and

```
  <InjectTailwind>
  <App />
  </InjectTailwind>
```
